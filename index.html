Veículos - Sistema de gestão de frotas do TRE-PB

Objetivo:

Auxiliar no desenvolvimento da solução para gestão de frotas do TRE-PB.

Tecnologias

  • Java EE 8;
  • JavaServer Faces(JSF) 2.2 com a extensão do Primefaces 3.5;
  • Java Persistense API(JPA) com Hibernate e Spring;
  • Oracle Database 12c;
  • JasperReports Framework para geração de relatórios com interface iReport;
  • Maven Project para controle de dependências;
  • Apache Subversion(SVN) como repositório central e para controle de versão;
  • NetBeans IDE;
  • Redmine para o gerenciamento dos projetos;
  • Tomcat 8.x.x como servidor web;
  • Uso das metodologias Scrum para gestão e planejamento dos projetos.

A equipe atualmente é composta por:

  • Gerente: Francisco José
  • Desenvolvedores: Rômullo (Supervisor técnico), Pedro e José Ricardo (Estagiários)
  • Product Owner: Mônica

Minhas atividades no estágio passarão por diversas etapas no processo de desenvolvimento, dentre elas:

  • Análise de requisitos
  • Descrição das histórias dos usuários
  • Modelagem
  • Desenvolvimento back-end e front-end
  • Documentação

O sistema deverá ser capaz de receber solicitações dos usuários e exibi-las em um painel onde o gestor deverá analisar os pedidos e montar as viagens adequando: rotas, disponibilidades dos motoristas e passageiros. A solução desenvolvida será Full-Stack.
